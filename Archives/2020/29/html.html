Releases

  • Conflict in Harmony has released. It allows you to bind multiple actions to one key.
  • Fabulously Optimized has released. It's a fabric modpack perfect for switching over from Optifine. It includes:
    • Sodium - optimizes general rendering to improve FPS, adds some graphics options (configurable)
    • Lithium - optimizes chunk loading
    • Phosphor - optimizes light rendering
    • LambDynamicLights - lets you light up the area by holding a torch and more (configurable)
    • Ok Zoomer - the zoom feature, as you'd expect (configurable)
    • motioNO - adds an option to disable dynamic FOV (disabled by default)
    • Jumploader - to make this modpack available on this site
    • Fabric API - makes most mods work
    • Mod Menu - gives you the list of mods and lets you adjust their configuration
    • Better Mod Button - puts the "mods" button where you'd expect
    • Not Enough Crashes - lets you play after a crash and report it comfortably
    • And more mods planned!
    • Hang on, that's all the mods that I use!
  • Topaz HUD has released. It adds a custom actionbar HUD displaying Health, Saturation. Elytra Durability, Coordinates and In-Game Time.
